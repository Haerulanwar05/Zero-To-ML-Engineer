# Zero-To-ML-Engineer

Informatics student's journey in Machine Learning. Exploring algorithms, data preprocessing, and building end-to-end ML projects from scratch.

Selamat datang di repositori dokumentasi belajar saya. Sebagai mahasiswa Teknik Informatika di Universitas Trilogi, saya membangun ruang ini sebagai bukti komitmen saya mendalami Machine Learning secara mandiri di luar kurikulum kampus. Fokus utama saya di sini adalah menjembatani jarak antara teori akademik dengan implementasi kode nyata yang saya pelajari melalui dokumentasi teknis dan berbagai resource industri di YouTube.

Perjalanan ini saya mulai dari fondasi paling dasar, yaitu manipulasi data menggunakan NumPy dan Pandas. Di dalam repositori ini, saya tidak hanya menyimpan kode yang sudah jadi, tetapi juga mencatat berbagai kendala teknis dan "Aha! moment" yang saya temui selama proses ngoding. Misalnya, saat saya berjibaku memahami logika axis pada Pandas atau saat mencoba mengoptimalkan efisiensi array pada NumPy. Saya percaya bahwa dokumentasi atas kesalahan dan proses pemecahan masalah jauh lebih berharga daripada sekadar hasil akhir yang sempurna.

Saat ini, saya sedang aktif bereksperimen di divisi Research and Science HIMATIKA, di mana saya mencoba menyelaraskan apa yang saya pelajari di sini dengan diskusi riset di organisasi. Target besar saya adalah menguasai alur kerja Machine Learning Engineering secara end-to-end agar siap berkarir sebagai Quantitative Analyst. Repositori ini akan terus berkembang seiring dengan bertambahnya pemahaman saya tentang algoritma yang lebih kompleks dan teknik deployment model di masa depan.
