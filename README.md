# Zero-To-ML-Engineer

Informatics student's journey in Machine Learning. Exploring algorithms, data preprocessing, and building end-to-end ML projects from scratch.

🤖 Project: Zero to ML Engineer

Selamat datang di repositori dokumentasi belajar saya. Sebagai mahasiswa Teknik Informatika di Universitas Trilogi, saya membangun ruang ini sebagai bukti komitmen saya dalam mendalami Machine Learning secara mandiri di luar kurikulum kampus. Fokus utama saya adalah menjembatani jarak antara teori akademik dengan implementasi praktis yang saya pelajari melalui dokumentasi teknis serta berbagai sumber industri.

Perjalanan ini saya mulai dari fondasi fundamental, yaitu manipulasi data menggunakan NumPy dan Pandas. Di dalam repositori ini, saya tidak hanya menyimpan kode akhir, tetapi juga mendokumentasikan proses penyelesaian masalah serta pemahaman teknis yang saya dapatkan selama tahap pengembangan. Sebagai contoh, saya mencatat bagaimana saya memahami logika koordinat pada struktur data Pandas atau teknik optimasi array pada NumPy untuk efisiensi komputasi. Saya percaya bahwa dokumentasi mengenai proses pemecahan masalah dan logika di balik kode jauh lebih berharga daripada sekadar hasil akhir yang tanpa penjelasan.

Saat ini, saya aktif bereksperimen di divisi Research and Science HIMATIKA, tempat saya menyelaraskan hasil belajar mandiri ini dengan diskusi riset di organisasi. Target jangka panjang saya adalah menguasai alur kerja Machine Learning Engineering secara menyeluruh untuk persiapan karir di bidang analisis data dan teknologi finansial. Repositori ini akan terus diperbarui seiring dengan perkembangan pemahaman saya mengenai algoritma yang lebih kompleks dan implementasi model di masa mendatang.
